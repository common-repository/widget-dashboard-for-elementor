== Description ==

Our dashboard will help you understand and track what Elementor widgets are being used (and where!). This can be very useful to determine which elements are needed, and which aren't, when trying to optimize performance across your Wordpress website.


== Installation ==

1. Upload the `widget-dashboard-elementor` folder to your `plugins` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. View your dashboard to see the widget usage information by navigating to `Tools > Widget Dashboard for Elementor`.
